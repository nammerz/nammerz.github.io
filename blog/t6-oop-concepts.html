Blocks, Procs and Lambdas

What the heck? Sounds Foreign

08 Mar 2015

Today's blog will be about Blocks, Procs and Lambdas. They sound absolutely confusing if you never heard of them but you probably already use them. You just didn't know that they were called that way.

We'll start off with the Procs

Procs are objects. Here is an example:

A proc is an instance of a Proc class.

p = Proc.new {puts “Hi everyone! I am a Proc!}

Procs in this example is the lower case “p”.

Other examples of procs:

p.call This will print out, “Hi everyone! I am a Proc!

p.class This will say 'Proc'

yummy_food = p yummy_food will now represent p

p

This will return a proc object

It isn't too hard to understand right? As you can tell by these examples, you probably already use them!

Blocks is part of a method call, or part of the syntax. It cannot be by itself or it will throw an error and can only be part of an argument.

{puts “Hi everyone! I am a Block!} This will throw an error

superman = {puts “Hi everyone! I am a Block!} This will also throw an error

[“Superman”, “Batman”, “Wolverine”].each {|x| puts x*2} This definitely works!

Now we get into the Lambdas! Something to know about them... they're Procs as well!

WHAT?!?!?!

p = Proc.new {puts “Hi everyone! I am a Proc!}

l = lambda { puts “Well, I'm a Lambda!}

proc.class returns 'Proc'

lam.class returns 'Proc'

There are differences though. Lambdas checks for all the argument in the method whereas Procs does not and will still run the program.

superman = lambda {|x| puts x} this lambda will take only 1 argument

superman.call(2) since there is only one argument, it will print out 2

superman.call there're no arguments stated so ArgumentError prints

superman.call(1,2,3) there're 3 but lambdas take specific # of arguments so an error prints

batman = Proc.new {|x| puts x} this proc will take only 1 argument

batman.call(2) since there is only one argument, it will print out 2

batman.call this will print nil

batman.call(1,2,3) this will print out “1” but not “2” or “3”

This is just a general explanation of Blocks, Procs, and Lambdas. It obviously gets much more complicated but we'll just leave it as this. I hope this explains just a tiny bit of what these three things talk about.
